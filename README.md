# Orbit Biplane Drag Optimization

Aerodynamic analysis and drag optimization of the Orbit biplane tail-sitter aircraft using OpenVSP for geometry development, FlightStream for aerodynamic simulations, and MATLAB-based implementation of Gudmundsson's drag prediction model.

## Objectives

* Analyze aerodynamic characteristics of the baseline Orbit configuration.
* Estimate lift, drag, and aerodynamic efficiency.
* Implement Gudmundsson's drag estimation methodology in MATLAB.
* Investigate design modifications for drag reduction and performance improvement.

## Tools

* OpenVSP
* FlightStream
* MATLAB

## Repository Structure

* `matlab/` : Drag and performance analysis codes
* `openvsp/` : Geometry models
* `flightstream/` : Simulation results
* `figures/` : Plots and visualizations
* `report/` : Reports and presentations
